# Telegram AI Bot с администрированием канала

Функции:
- Ответы на сообщения через DeepSeek AI
- Админ-команды (доступны только вам, ID=682446170)
- Отправка постов в канал (/post)
- Закрепление/открепление сообщений (/pin, /unpin)
- Бан/разбан пользователей в группе (/ban, /unban)
- Проверка подписки (/check_sub)
- Автоматическая модерация (удаление ссылок и запрещённых слов)
- Обязательная подписка на канал (опционально)

## Настройка перед запуском
1. Укажите ID вашего канала в файле `config.py` (CHANNEL_ID)
2. Если используете группу для модерации, укажите GROUP_ID
3. При необходимости настройте REQUIRE_SUBSCRIPTION и FORBIDDEN_WORDS

## Деплой на Bothost
- Загрузите все файлы в Git или через ZIP
- Добавьте переменные окружения: TELEGRAM_TOKEN, DEEPSEEK_API_KEY, CHANNEL_ID
- Убедитесь, что бот добавлен в канал как администратор

## Команды администратора
- /admin_stats – статистика бота
- /post текст – отправить пост в канал
- /pin message_id – закрепить сообщение
- /unpin – открепить
- /ban user_id – забанить в группе
- /unban user_id – разбанить
- /check_sub user_id – проверить подписку
