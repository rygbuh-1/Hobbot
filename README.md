# Telegram AI Bot на DeepSeek (исправленная версия)

## Важно
Бот работает в режиме **long polling**, поэтому **не требует домена и вебхука**. 
При развертывании на Bothost **выключите опцию "Использовать домен"**, если это возможно.
Если платформа требует домен — оставьте его включенным (health-сервер будет работать).

## Переменные окружения (необязательно, значения уже в коде):
- `TELEGRAM_TOKEN`
- `DEEPSEEK_API_KEY`
- `PORT` (по умолчанию 8080)

Исправлена ошибка `proxies` за счёт обновления зависимостей.
