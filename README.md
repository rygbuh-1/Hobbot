# Telegram AI Bot на DeepSeek

Этот бот отвечает на сообщения с помощью нейросети DeepSeek.

## Как развернуть на Bothost

1. Создайте новый проект на Bothost.
2. Выберите тип "Telegram Bot".
3. В поле "Git URL" укажите URL этого репозитория (или загрузите файлы вручную).
4. **ВАЖНО**: отключите опцию "Использовать домен", так как бот работает в режиме long polling (не требует вебхука). 
   - Если платформа настаивает на домене, включите его — health-сервер всё равно будет отвечать на /health.
5. В разделе "Переменные окружения" (Environment Variables) добавьте:
   - `TELEGRAM_TOKEN` = ваш токен от BotFather
   - `DEEPSEEK_API_KEY` = ваш ключ DeepSeek
   (Значения уже вставлены в код по умолчанию, но лучше задать их через env для безопасности.)
6. Нажмите "Deploy".

Бот автоматически запустится и будет отвечать на сообщения.

## Локальный запуск

```bash
pip install -r requirements.txt
python app.py
