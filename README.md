## Команды администратора
- /admin_stats — проверить настройки
- /post текст — отправить пост в канал
- /pin ID — закрепить сообщение в канале
- /unpin — открепить
- /ban ID — забанить пользователя в группе
- /unban ID — разбанить
- /check_sub ID — проверить подписку на канал

## Функции
- Автоматическая модерация в группе (удаление ссылок и запрещённых слов)
- Обязательная подписка на канал перед общением с ботом
- Ответы через DeepSeek AI в личных сообщениях
