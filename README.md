# Полнофункциональный бот для администрирования канала и группы

- Работает на основе проверенного минимального бота
- Админ-команды: /admin_stats, /post, /pin, /unpin, /ban, /unban, /check_sub
- Модерация группы (удаление ссылок и запрещённых слов)
- Ответы через DeepSeek AI в личных сообщениях
- Проверка подписки отключена для простоты

## Развёртывание
Замените файлы на Bothost этим ZIP-архивом, перезапустите бота.
