# Бот с мониторингом канала

Бот:
- Отвечает на сообщения в группе (как и раньше)
- Автоматически комментирует новые посты в указанном канале и отправляет комментарий в группу

## Настройка
- Убедитесь, что бот добавлен в канал как **администратор** (иначе не получит события channel_post)
- В коде заданы:
  - MONITOR_CHANNEL_ID = -1003154677228
  - TARGET_GROUP_ID = -1002688844179

## Команды админа (в личке)
- /toggle_monitor – вкл/выкл комментирование
- /admin_stats – статус
