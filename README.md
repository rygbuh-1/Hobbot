# Бот-эксперт по ставкам с авто-поиском

Бот самостоятельно ищет информацию в Google (травмы, составы, новости) перед тем, как дать анализ матча.

## Важное предупреждение
- Google может блокировать запросы (капча). Если поиск не работает, отключите его командой `/toggle_search` (только админ) и предоставляйте ссылки вручную.
- Для стабильной работы рекомендуется использовать платный API (например, SerpAPI), но в рамках теста оставляем бесплатный метод.

## Команды
- `/toggle_search` – включить/выключить автоматический поиск
- `/clear_history` – очистить историю
- `/admin_stats` – статус

## Деплой
Загрузите архив на Bothost, перезапустите бота. Убедитесь, что добавлены переменные окружения TELEGRAM_TOKEN и DEEPSEEK_API_KEY.
