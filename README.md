# Финальная версия бота

Бот отвечает на **любые** текстовые сообщения в группе (кроме команд).  
Логирование каждого сообщения для отладки.

## Команды
- `/start` – приветствие
- `/help` – справка
- `/test` – проверка работоспособности
- `/admin_stats` – статус (только для администратора 682446170)
- `/toggle_chat` – включить/выключить общение в группах

## Настройки BotFather
- **Group Privacy** → выключить (Turn off)
- **Allow Groups** → включено

## Деплой
Загрузить все файлы на Bothost, перезапустить сервис.
