# Бот-эксперт по ставкам с авто-поиском (SerpAPI)

Бот самостоятельно ищет информацию в интернете через SerpAPI (Google), анализирует матчи по 5-шаговой методологии и даёт прогнозы. Все утверждения подкрепляются ссылками на источники.

## Функции
- Автоматический поиск новостей, травм, составов.
- Память диалога (сохраняется в файл `history.json`).
- Мониторинг канала и комментирование постов в группе.
- Работа в группе и личных сообщениях.

## Команды (администратор 682446170)
- `/toggle_monitor` – включить/выключить комментирование канала.
- `/admin_stats` – статистика.
- `/clear_history` – очистить историю для текущего пользователя.

## Переменные окружения (необязательно, ключи уже в коде)
- `TELEGRAM_TOKEN`
- `DEEPSEEK_API_KEY`
- `SERPAPI_KEY`

## Деплой на Bothost
1. Загрузить все файлы (bot.py, requirements.txt, Dockerfile).
2. Перезапустить сервис.
3. Убедиться, что бот добавлен в канал и группу, Group Privacy выключена в BotFather.
