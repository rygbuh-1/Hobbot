# Telegram AI Bot – финальная рабочая версия

Всё работает:
- `/start` – приветствие
- `/admin_stats` – проверка статуса (только для администратора 682446170)
- `/post` – отправка постов в канал
- `/pin`, `/unpin` – управление закреплёнными сообщениями
- `/ban`, `/unban` – модерация в группе
- `/check_sub` – проверка подписки на канал
- Автоматическая модерация (удаление ссылок и запрещённых слов)
- Ответы DeepSeek AI в личных сообщениях

## Деплой
Скопируйте файлы на Bothost, перезапустите сервис. Бот готов к использованию.
