# Бот-эксперт по ставкам с авто-поиском и защитой от дублей

## Исправления
- Использована новая версия openai (1.58.1) и httpx (0.27.2) для устранения ошибки 'proxies'.
- Защита от дублей на посты канала: каждый post_id запоминается с таймстампом и не обрабатывается повторно в течение 60 секунд.
- Ответы стали короче (макс 2500 символов), чтобы уменьшить вероятность разбиения.

## Деплой
Полностью заменить файлы на Bothost, перезапустить бота.
