# Бот-эксперт по ставкам на спорт

Бот помогает анализировать матчи по методологии из 5 шагов, но **не ищет информацию сам**. Он запрашивает у пользователя актуальные источники (новости о травмах, составы, пресс-конференции) и только на их основе даёт прогноз.

## Как использовать
- Напишите боту название матча и приложите ссылки на источники.
- Бот проведёт анализ по шагам.
- Если данных недостаточно, бот попросит их.

## Команды
- `/clear_history` – очистить историю диалога
- `/admin_stats` – статистика (админ)
- `/toggle_monitor` – вкл/выкл комментирование постов из канала

## Деплой
Загрузите архив на Bothost, перезапустите бота. Убедитесь, что добавлены переменные окружения TELEGRAM_TOKEN и DEEPSEEK_API_KEY.
